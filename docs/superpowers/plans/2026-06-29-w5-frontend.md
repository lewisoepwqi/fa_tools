# W5 前端收口与分页 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 收口 FA Tools 前端剩余缺口——列表分页、preview-rows 分页接入、日记账动态列、规则拖拽重排、公司选择器修复、权限门控补齐，并补 `GET /api/companies` 与修 reorder 审计公司归属。

**Architecture:** 后端沿用既有 `Page[T]` 分页信封（`limit`/`offset` + `{items,total,limit,offset}`）把扁平 list 端点改为分页；扩展 `ConversionRunSummary` 分状态计数支撑分页下的统计；新增平台共享 `GET /api/companies`。前端 list api 改返回 `Page<T>`，列表页接服务端分页；批次详情改用 preview-rows 分页端点并按日记账模板 `columns_json` 动态渲染列；规则页加拖拽重排；公司选择器修 admin/'all' 场景。

**Tech Stack:** 后端 FastAPI + SQLAlchemy 2 + Pydantic v2 + pytest；前端 React 18 + TypeScript + Ant Design v5 + axios + Vite + Playwright。

## Global Constraints

- 后端测试门禁：`cd backend && .venv/bin/pytest -q` 全绿；lint：`cd backend && .venv/bin/python -m ruff check .` 全绿（**本环境 ruff 在 .venv 内，用 `python -m ruff`，PATH 无 ruff**）。

- 分页约定（**逐字沿用**）：`limit: int = Query(100, ge=1, le=500)`、`offset: int = Query(0, ge=0)`；返回 `Page[T]{items, total, limit, offset}`（`app/tools/bank_journal/schemas/pagination.py`）。
- 语言：代码注释、提交信息、UI 文案一律中文；技术术语保留原文。
- 版本化不可变：不得改动既有版本行；不在本工作流重建已存在的编辑/版本/启停端点。

```python
from fastapi import APIRouter, Depends
from app.api.deps import DbSession, CurrentUserDep, require, accessible_company_filter
from app.core.permissions import Permission
from app.models.company import Company
from pydantic import BaseModel

router = APIRouter(prefix="/api/companies", tags=["companies"])

class CompanyResponse(BaseModel):
    id: str
    name: str

@router.get("", response_model=list[CompanyResponse], dependencies=[Depends(require(Permission.READ))])
def list_companies(db: DbSession, user: CurrentUserDep) -> list[CompanyResponse]:
    accessible = accessible_company_filter(user)
    q = db.query(Company)
    if accessible is not None:
        q = q.filter(Company.id.in_(accessible))
    return [CompanyResponse(id=c.id, name=c.name) for c in q.order_by(Company.name).all()]
```
